Создать 4-и потока, каждый из них выводит 5 раз строку.
Первый поток выводит 5-ть раз 'Квас для вас', второй 5-ть раз ' и нас, ',
третий 5-ть раз 'ядреный, аж слезы', четвертый 5-ть раз ' из глаз! '

Нужно настроить работу потоков так, чтобы после запуска всех потоков, в
консоли было выведено:
'Квас для вас и нас, ядреный, аж слезы из глаз!'
Т.е. первый печатает одну подстроку, потом второй - вторую подстроку, затем
третий - третью подстроку и т.д. до полного предложения. Далее все тоже,
с новой строки и так 5-ть раз.

Использовать только методы *.wait() и *.notifyAll().
